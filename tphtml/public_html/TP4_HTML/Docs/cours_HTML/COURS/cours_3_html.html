HTML 5 – Partie 3

Introduction

Certains éléments en HTML ne vont être constitués que d’une balise qu’on appelle alors orpheline. Cela va être le cas pour certains éléments qui ne possèdent pas de contenu textuel, comme l’élément <br> qui sert simplement à créer un retour à la ligne en HTML. C'est le cas de la balise <img> qui n'a pas pour vocation de contenir du texte par définition.

Voici quelques exemples :

<br />
<hr />
<img … /> Run

Les images : <img>

La balise <img>

L’insertion d’une image se fait avec la balise <img />.

C’est une balise unique de type en ligne, ce qui signifie que les images s’enchaînent en fonction de leurs dimensions.

La syntaxe de base est :

<img src="chemin relatif ou absolu vers l'image"> <!-- le / de fin n'est pas obligatoire -->
<img src="chemin relatif ou absolu vers l'image" /> Run

  • L’attribut src indique que l’image provient d’un fichier externe dont l’adresse est donnée.

Exemples :

<img src="http://www.html5.com/images/image.png" /><a href="lien">
<img src=".image/img1.png" /></a>
<img src="image.png" /> Run

Si le lien vers l'image est invalide, une image cassée s'affichera :

  • Les attributs height et width permettent de définir la hauteur et la largeur en pixels. Sans précisions, l’image garde ses dimensions originales.

Pour une image destinée à un fond de site (occupant 100% de la page), il faut trouver un compromis entre taille/poids et qualité. Une résolution recommandée est 1920 x 1080px.

<img src="./images/img1.png" width="100px" height="100px" /> Run

Si l'image dépasse la résolution de l'écran, des barres de défilement apparaîtront.

Une image trop petite agrandie perdra en netteté.

Exemple :

<img src="paysage3.jpeg" alt="" width="800px" height="500px" />
<img src="paysage2.jpeg" alt="" width="800px" height="500px" /> Run

Pour obtenir une image qui prend toute la largeur et s'adapte à l'écran, utilisez le pourcentage pour la largeur.

Exemple :

<img src="paysage2.jpeg" alt="" width="100%" /> Run

  • L’attribut alt est obligatoire et doit contenir une description de l'image.

  • L'attribut title affiche une infobulle lors du survol.

Les formats et poids des images

Pour un chargement rapide, utilisez des formats compressés : GIF, JPG, PNG.

  • GIF : 256 couleurs, animation, transparence possible.

  • JPG : 16.7 millions de couleurs, compression avec perte, idéal pour les photos.

  • PNG : compression sans perte, supporte la transparence.

  • Le BMP est à éviter, il est trop lourd.

Une photo peut peser entre 4 et 8 Mo. Il est conseillé de réduire son poids avec un logiciel de retouche.

Remarque : Une image de 1920 x 1080 px peut peser 700 ko. Même si l'affichage est réduit, le poids reste identique.

<img src="paysage2.jpeg" alt="" width="100px" /> Run

Optimisez vos images pour économiser votre espace serveur.

La balise <figure> et <figcaption>

L’élément <figure> permet d’insérer des médias et <figcaption> fournit une légende pour apporter de la sémantique.

<figure>
  <img src="./images/img1.png" alt="Paysage de montagne" title="paysage de montagne"/>
  <figcaption>un paysage de montagne</figcaption>
</figure> Run

Les styles sur image

Image d'arrière-plan

La propriété background-image permet d’ajouter une image en fond d’un élément, utilisée pour des images purement décoratives.

La propriété background-repeat contrôle la répétition (repeat, repeat-x, repeat-y, no-repeat).

La propriété background-size redimensionne l'image (contain, cover ou dimensions spécifiques).

La propriété background-position définit la position initiale de l'image.

La propriété background-attachment définit si l'image est fixe dans la zone d'affichage.

L'opacité d'une image

La propriété opacity gère la transparence d'une image (valeurs de 0 à 1). Même à 0, l'image reste présente dans la mise en page.

Exemple : en survol, l'opacité passe de 1 à 0.5 :

img.opacite {
   opacity: 1;
}
img.opacite:hover {
   opacity: 0.5;
} Run

Même une image avec opacity 0 reste un bloc dans la page.

Les dimensions d'une image

Les attributs width et height permettent de redimensionner une image. Si l'un est défini, l'autre s'adapte pour conserver les proportions.

L'utilisation du % permet à l'image de s'adapter à la résolution de l'écran.

Exemple :

<img src="paysage2.jpeg" alt="" width="100%"/> Run

  • L’attribut alt est obligatoire et doit décrire l'image.

  • L’attribut title affiche une infobulle lors du survol.

Les formats et poids des images

Utilisez des formats compressés (GIF, JPG, PNG) pour réduire le temps de chargement.

  • GIF : 256 couleurs, animation, transparence possible.

  • JPG : 16.7 millions de couleurs, compression avec perte, idéal pour les photos.

  • PNG : compression sans perte, supporte la transparence.

Une photo peut peser entre 4 et 8 Mo. Réduisez-la avec un outil de retouche.

Remarque : Une image de 1920 x 1080 px peut peser 700 ko. Même si affichée à une taille réduite, le poids reste le même.

<img src="paysage2.jpeg" alt="" width="100px"/> Run

Optimisez vos images pour économiser votre espace serveur.
