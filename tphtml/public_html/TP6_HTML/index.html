Exercice 1 : Reproduire un tableau

Écrire le code HTML permettant de reproduire la structure du tableau suivant (HTML en premier, puis la mise en forme en CSS).

Code HTML :

<table>
  <thead>
    <tr>
      <th>Colonne 1</th>
      <th>Colonne 2</th>
      <th>Colonne 3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ligne 1, Cellule 1</td>
      <td>Ligne 1, Cellule 2</td>
      <td>Ligne 1, Cellule 3</td>
    </tr>
    <tr>
      <td>Ligne 2, Cellule 1</td>
      <td>Ligne 2, Cellule 2</td>
      <td>Ligne 2, Cellule 3</td>
    </tr>
  </tbody>
</table>

Aperçu :

Exemple de tableau
Colonne 1 Colonne 2 Colonne 3
Ligne 1, Cellule 1 Ligne 1, Cellule 2 Ligne 1, Cellule 3
Ligne 2, Cellule 1 Ligne 2, Cellule 2 Ligne 2, Cellule 3
