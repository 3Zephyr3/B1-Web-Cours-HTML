Exercice 2 : Reproduire le zoning d'une page

Reproduire le zoning d'une ou plusieurs pages. Dans cet exercice, on ne s'intéresse qu'à la structure (le zoning) de la page et non à son contenu.
Réaliser un fichier CSS séparé et une page web pour chaque exercice, en veillant à ce que la page garde ses proportions.
