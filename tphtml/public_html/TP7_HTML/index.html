Exercice 1

Dans cet exercice, nous avons utilisé l’inspecteur de code pour repérer et comprendre les différentes zones qui structurent la page web. Objectif : identifier clairement le zoning du site (par exemple : <header>, <nav>, <aside>, <article>, <footer>).

Sur la page j-willette.developpez.com, nous avons trouvé par exemple :

  • <header> : en-tête du site
  • <nav> : menu de navigation principal
  • <aside> : menu latéral
  • <article> : contenu principal
  • <footer> : pied de page

La partie <article> inclut titres, paragraphes, listes, liens, images, etc.
