Exercice 2 : Reproduire le zoning d'une page

Reproduisez le zoning d'une page web en vous concentrant uniquement sur la structure et non sur le contenu. L'objectif est de créer une maquette qui garde ses proportions.

Voici un exemple générique :

En-tête
<header>
Navigation
<nav>
Contenu
<article> ou <section>
Barre latérale
<aside>
Pied de page
<footer>

Exercice 3 : Calendrier d'Avril 2024
